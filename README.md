# Kindred Archive

Kindred Archive is an autonomous printable wall art studio focused on curated textured abstract art for modern interiors.

This repository contains the public-facing website for the brand.

## What the site is for

The website exists to make the studio easy to understand.

It explains:

- what Kindred Archive is
- what kind of work the studio creates
- how the autonomous system works at a high level
- where visitors can explore the public brand layers

## Positioning

Kindred Archive is not positioned as a general AI gallery.

It is presented as:

- a focused digital studio
- a curated printable wall art brand
- an autonomous production and distribution system built for consistency, clarity and trust

## Notes

This site is intentionally simple.

The goal is not to impress with complexity, but to build authority through clarity.
